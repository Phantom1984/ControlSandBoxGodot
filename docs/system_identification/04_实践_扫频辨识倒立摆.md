# 04 实践：扫频辨识一阶倒立摆

## 1. 实验目标

通过扫频激励辨识一阶倒立摆"小车力 → 摆角"通道的频率响应，得到伯德图与相干图，验证与理论传递函数的吻合度。

## 2. 理论参考模型

小车-倒立摆在小角度（`θ ≈ 0`）处线性化：

```
G(s) = Θ(s) / F(s) = 1 / (M·l·s² - (M+m)·g)
```

**特征**：
- 双极点，一正一负（不稳定系统）
- 直流增益 `G(0) = -1/((M+m)·g)`
- 不稳定极点 `s = +√((M+m)g / (Ml))`

**核心挑战**：开环不稳定，无法直接扫频，必须**在闭环下辨识**。

## 3. 闭环辨识方案

```
                 ┌──────┐
chirp u ──(+)──►│ C(s) │──► f ──► Plant ──► θ
           ▲     └──────┘                  │
           │                               │
           └───────────────────────────────┘
```

激励信号叠加到控制器参考输入，测量 `θ` 与 `u`（注入信号本身，而非控制力 `f`）。

**估计的传递函数**（闭环）：
```
Ĝyu(s) = Θ / U = C(s)·G(s) / (1 + C(s)·G(s))
```

再通过代数反推得到开环 `G(s)`：
```
G(s) = Ĝyu(s) / (C(s)·(1 - Ĝyu(s)))
```
其中 `C(s)` 为已知的 PID 控制器频率响应（解析可算）。

## 4. 实验参数

| 参数 | 值 | 说明 |
|------|----|----|
| 物理采样率 | 60 Hz | Godot physics tick |
| 扫频范围 | 0.1 ~ 5 Hz | 覆盖主模态 |
| 扫频时长 | 60 s（建议 120~180 s） | 频率分辨率 ~0.017 Hz |
| 激励幅值 | 0.02 rad | 摆角偏置，保证小角度线性 |
| Welch 段长 `L` | 1024 | 配合 60Hz 采样需较长数据 |
| 重叠 | 50% | Hann 窗 |
| 段数 `K` | ~7（120s 数据） | 越多越好，至少 ≥ 8 |

### 4.1 采样率陷阱

Godot physics 60Hz 采样，1024 点 FFT 意味着每段 17 秒。60 秒数据仅能分 ~3 段，**统计意义不足**。

**解决方案（按推荐度排序）**：
1. **延长录制时长** 到 120~180 秒，得到 7~10 段
2. **离线辨识**：录制后用更高 FFT 点数处理，不依赖实时
3. **过采样**：物理帧之间用上一帧状态插值，提升有效采样率（增加冗余信息，不提高真实分辨率）

## 5. 实验流程

1. **预调**：启动 PID 稳定倒立摆（kp/ki/kd 调好，摆角稳定在 0 附近）
2. **零扰动**：关闭环境面板中的摩擦/阻力/周期扰动（避免污染相干）
3. **注入激励**：启动 Chirp 源，叠加到控制器参考摆角
4. **录制**：DataRecorder 同步录制 `u(t)`、`θ(t)`（建议同时录 `x(t)` 用于诊断）
5. **谱估计**：Welch 法计算 `Suu`、`Sθθ`、`Suθ`
6. **FRF 估计**：H1 估计 `Ĥ1(f) = Suθ / Suu`
7. **相干**：`γ²(f) = |Suθ|² / (Suu·Sθθ)`
8. **画图**：伯德图（幅频 + 相频）+ 相干图
9. **对比**：与理论闭环传递函数对比，验证吻合度
10. **（可选）反推**：从闭环 `Ĝyu` 反推开环 `G`，对比解析模型

## 6. 结果分析

### 6.1 期望的伯德图特征（闭环）

- **低频**：增益接近 1（闭环跟踪良好）
- **带宽处**：-3 dB 点，对应系统响应速度
- **高频**：增益衰减（控制器带宽外）
- **相位**：从 0° 滞后到 -180°，可能进一步到 -360°（二阶以上）

### 6.2 相干函数判读

- **带宽内**：`γ² > 0.9`（线性关系好，估计可信）
- **高频段**：`γ²` 下降（SNR 低 + 离散化影响）
- **共振峰处**：若 `γ²` 突降，可能存在非线性或分辨率不足

### 6.3 误差诊断表

| 现象 | 可能原因 | 解决方法 |
|------|---------|---------|
| 相干全频段偏低 | 激励幅值过小 / 噪声大 | 增大幅值 30%，检查环境扰动是否关掉 |
| 高频段相干骤降 | 采样率不足 / 量化误差 | 限制分析频率上限到 fs/4 |
| 共振峰处 H1、H2 差异大 | 分辨率不足 / 非线性 | 加大段长 L，或换步进正弦 |
| 多次实验不一致 | 系统非平稳 | 检查环境参数、初始条件 |
| 低频段相干低 | 段长不足 | 延长录制时长 |

## 7. 模型反推（可选深度分析）

从闭环 `Ĝyu(f)` 反推开环 `G(f)`：
```
G(f) = Ĝyu(f) / (C(f)·(1 - Ĝyu(f)))
```

**PID 控制器的频率响应**（解析）：
```
C(jω) = kp + ki/(jω) + kd·jω
```

**注意**：
- 闭环带宽外的 `Ĝyu` 估计噪声大，反推会放大误差
- 建议只在相干 `γ² > 0.8` 的频段做反推
- 反推结果与解析 `G(s) = 1/(Mls² - (M+m)g)` 对比时，关注极点位置

## 8. 验证基准

| 验证项 | 方法 |
|-------|------|
| FRF 形状 | 与解析闭环传递函数对比 |
| 带宽 | 与 PID 参数推算的 -3dB 频率对比 |
| 相干 | 带宽内应 > 0.9 |
| 多次重复 | 同参数下重复 3 次，估计方差应 < 5% |
| 模型预测 | 用辨识模型预测另一组激励下的响应，对比实测 |
