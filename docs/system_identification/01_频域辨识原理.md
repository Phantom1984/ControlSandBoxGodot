# 01 频域辨识原理

## 1. 引言

频域辨识的核心思想：通过测量系统在频域上的输入输出关系，直接估计传递函数（频率响应），无需构建参数化模型。

**优点**：
- 非参数估计，不预设模型阶数
- 直观可视化（伯德图）
- 与噪声特性解耦（在 H1/H2 假设下）

**适用场景**：
- 控制器设计前的被控对象建模
- 共振峰、带宽的直观判断
- 模型验证

## 2. LTI 系统的频域描述

### 2.1 频率响应函数 (FRF)

对线性时不变系统（LTI）：
$$
y(t) = h(t)*u(t)+n(t) \tag{1.1}
$$
其中$$h(t)$$为系统冲激响应，$$n(t)$$为噪声。

傅里叶变换到频域：
$$
Y(f)=H(f)\cdot U(f)+N(f) \tag{1.2}
$$
其中 $$H(f)$$即真实传递函数（频率响应）。

### 2.2 功率谱密度关系

- 自谱：$$S_{xx}(f)=E[|X(f)|^2]$$
- 互谱：$$S_{xy}(f)=E[X^*(f)\cdot Y(f)]$$（$$^*$$表示共轭）

理想无噪声情况下：
$$
H(f)=\frac{S_{uy}(f)}{S_{uu}(f)} \tag{1.3}
$$

## 3. 噪声模型与三种估计器

实际系统噪声不可避免，根据噪声注入位置不同，需采用不同的估计器：

### 3.1 输出噪声模型 → H1 估计

```
u(t) ──► H ──► (+) ──► y(t)
                 ▲
                 n(t)
```
假设$$Y=H\cdot U+N$$ ，且 $$U$$与 $$N$$不相关。
$$
\hat H(f)=\frac{S_{uy}(f)}{S_{uu}(f)} \tag{1.4}
$$
**性质**：
- 在 $$U$$、$$N$$ 不相关时为无偏估计
- 抑制输出端噪声
- 工程最常用，本项目默认采用

### 3.2 输入噪声模型 → H2 估计

```
u(t) ──► (+) ──► H ──► y(t)
          ▲
          n(t)
```
测量到的 `um = u + n`，假设 `u` 与 `n` 不相关。

```
Ĥ2(f) = Syy(f) / Syu(f) = Syy(f) / Suy*(f)
```

**性质**：
- 在输入噪声独立时为无偏估计
- 抑制输入端噪声
- 关系：`|Ĥ2| ≥ |Ĥ1|`（当相干 < 1 时）

### 3.3 双侧噪声 → Hv 估计

输入输出均有噪声时，按噪声方差比加权。本质是广义特征值问题。

**性质**：
- 估计值介于 H1 和 H2 之间
- 需已知噪声方差比
- 实际工程少用

## 4. 相干函数 (Coherence)

```
γ²(f) = |Suy(f)|² / (Suu(f) · Syy(f))    ∈ [0, 1]
```

**物理意义**：输出中能由输入线性解释的比例。

| γ² | 解读 |
|----|------|
| = 1 | 完全线性关系，无噪声 |
| > 0.9 | 高质量，估计可信赖 |
| 0.5 ~ 0.9 | 有噪声或轻度非线性 |
| < 0.5 | 噪声主导 / 非线性强 / 频率分辨率不足 |

**与 H1/H2 的关系**：
```
|Ĥ1|² = γ² · |Ĥ2|²
```
即 H1 总是不大于 H2，相干越接近 1，两者越一致。

## 5. 估计质量评估

- **相干函数**：频点级可信度指标
- **H1 vs H2 比值**：诊断噪声方向（输入侧 vs 输出侧）
- **多次平均的方差**：判断是否收敛
- **共振峰处**：相干下降通常意味着分辨率不足或非线性

## 6. 离散实现要点

### 6.1 单边谱与双边谱
实信号 FFT 结果对称，工程上通常只取正频率半段，并乘以 2（除直流外）。

### 6.2 频率分辨率
```
Δf = fs / N
```
其中 `fs` 为采样率，`N` 为 FFT 点数。`Δf` 决定可分辨的最小频率间隔。

### 6.3 与 Welch 法的关系
直接用整段数据算 `Sxx = |FFT(x)|²` 是周期图法，方差极大。Welch 法通过分段加窗平均显著降低方差，是工程上估计 `Sxx/Syy/Sxy` 的标准做法，详见 [02_Welch法.md](./02_Welch法.md)。

## 8. 关键公式速查

| 项目 | 公式 |
|------|------|
| 自谱 | `Sxx(f) = E[|X(f)|²]` |
| 互谱 | `Sxy(f) = E[X*(f)·Y(f)]` |
| H1 估计 | `Ĥ1 = Suy / Suu` |
| H2 估计 | `Ĥ2 = Syy / Suy*` |
| 相干 | `γ² = |Suy|² / (Suu·Syy)` |
| H1-H2 关系 | `|Ĥ1|² = γ²·|Ĥ2|²` |
