# 05 时域辨识（ARX 最小二乘）

> 本篇为阶段 2 占位骨架，频域辨识完成后展开。

## 1. 为什么需要时域辨识

频域辨识给出非参数 FRF，时域辨识直接得到**参数化模型**（差分方程系数），可直接用于：
- LQR / MPC 控制器设计（需状态空间模型）
- 数字控制器离散设计
- 模型预测与仿真

## 2. ARX 模型

```
A(z⁻¹)·y(k) = B(z⁻¹)·u(k-d) + e(k)
```

其中：
- `A(z⁻¹) = 1 + a₁·z⁻¹ + … + aₙₐ·z⁻ⁿᵃ`
- `B(z⁻¹) = b₁·z⁻¹ + … + bₙᵦ·z⁻ⁿᵇ`
- `d` 为纯延迟（采样周期数）
- `e(k)` 为白噪声

**注**：ARX 假设噪声与输入同通道进入，对噪声模型有较强假设。若噪声特性不符，需用 ARMAX / OE / BJ 等更一般模型。

## 3. 最小二乘解

将模型写成线性回归形式：
```
y(k) = φᵀ(k)·θ + e(k)
```

**回归向量**：
```
φ(k) = [-y(k-1), …, -y(k-nₐ), u(k-d-1), …, u(k-d-nᵦ)]ᵀ
```

**参数向量**：
```
θ = [a₁, …, aₙₐ, b₁, …, bₙᵦ]ᵀ
```

**最小二乘解**：
```
θ̂ = (Φᵀ·Φ)⁻¹·Φᵀ·Y
```

**依赖**：`scripts/linalg/Matrix.gd` 实现矩阵求逆、解方程。

## 4. 阶次确定

| 方法 | 原理 | 优点 | 缺点 |
|------|------|------|------|
| **AIC** | `AIC = N·ln(σ²) + 2p` | 平衡拟合与复杂度 | 倾向于选高阶 |
| **BIC** | `BIC = N·ln(σ²) + p·ln(N)` | 比 AIC 更严格 | 倾向于选低阶 |
| 损失函数法 | 随阶次增加，损失下降变缓处为最优 | 直观 | 主观 |
| 零极点分析 | 检查冗余极零点对 | 物理意义强 | 需人工判断 |

其中 `p` 为参数个数，`σ²` 为残差方差，`N` 为数据长度。

## 5. 模型验证

### 5.1 残差白噪声检验
- 残差自相关函数应接近 δ 函数
- 残差与输入的互相关应接近 0
- 否则模型未捕获全部动态

### 5.2 交叉验证
- 用未参与辨识的数据集验证
- 模型预测 vs 实测对比

### 5.3 阶跃响应对比
- 辨识模型与实测阶跃响应叠加
- 检查稳态增益、时间常数、超调

## 6. 与频域辨识的关系

ARX 模型对应的离散传递函数：
```
G(z) = B(z⁻¹) / A(z⁻¹) · z⁻ᵈ
```

可转换为：
- **连续传递函数**：双线性变换 / 零阶保持等价
- **频率响应**：代入 `z = e^(jωTₛ)`
- 与频域辨识结果叠加对比，相互验证

## 7. 与控制器的对接

辨识得到的离散模型可直接用于：
- **LQR**：转状态空间，求 Riccati 解
- **MPC**：作为预测模型
- **ARDC**（自适应控制）：在线辨识 + 控制律更新

## 9. 风险与注意

- **数值条件数**：`ΦᵀΦ` 可能病态，需 SVD 或正则化
- **激励充分性**：辨识数据必须有持续激励（详见 [03_激励信号设计](./03_激励信号设计.md)）
- **闭环辨识**：与频域辨识类似，倒立摆需在闭环下辨识，需用闭环辨识方法（如 IV、间接法）

## 10. 进阶模型（远期）

| 模型 | 方程 | 适用 |
|------|------|------|
| ARMAX | `A·y = B·u + C·e` | 噪声有色的通用场景 |
| OE（输出误差） | `y = B/F·u + e` | 噪声仅加在输出端 |
| BJ（Box-Jenkins） | `y = B/F·u + C/D·e` | 噪声模型最灵活 |
| 状态空间 | `x(k+1)=Ax+Bu, y=Cx+Du` | 多变量、MPC 用 |
