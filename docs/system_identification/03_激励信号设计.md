# 03 激励信号设计

## 1. 激励信号的要求：持续激励

辨识的前提是输入信号充分"激发"系统的所有动态模态。

**n 阶持续激励 (PE-n)**：输入功率谱在至少 n 个频率点上有非零能量，且对应 DFT 矩阵满秩。

**实际含义**：要辨识 n 阶系统，激励信号至少需在 n 个频率上含能量。宽带信号天然满足高阶 PE。

## 2. 常见激励信号对比

| 信号 | 频谱 | 优点 | 缺点 | 适用 |
|------|------|------|------|------|
| 阶跃 | 宽带衰减 | 简单 | SNR 低，能量分布不均 | 一阶系统快速辨识 |
| **步进正弦** | 离散线谱 | SNR 极高，精度高 | 慢，逐频点 | 高精度频响测量（用户公司） |
| **Chirp** | 宽带平缓 | 快，频段可控 | 边缘瞬态泄漏 | 频域辨识主力 |
| PRBS | 离散平谱 | 白噪近似 | 需高频采样 | 时域辨识主力 |
| 多正弦 | 离散线谱 | 能量集中，峰值因数低 | 设计复杂 | 高 SNR 频域辨识 |

## 3. 步进正弦 Stepped Sine（重点）

逐个频率点驻留，每个频率测一组稳态响应：
```
u(t) = A · sin(2π · fk · t),    t ∈ [tk, tk + Tk]
```

### 3.1 每个频点测量流程

1. 输入正弦波，频率 `fk`
2. 等待瞬态衰减（典型 3~5 倍系统时间常数）
3. 录制稳态响应 `Tk` 秒
4. **同频解调**：用同频正弦/余弦与响应相乘 + 积分，提取同相分量 `I`、正交分量 `Q`
5. 计算该频点的增益与相位：
   ```
   H(fk) = (I + j·Q) / A
   |H| = √(I² + Q²) / A
   ∠H = atan2(Q, I)
   ```

### 3.2 优点

- 每个频点 SNR 极高（窄带滤波效果）
- 频率密度可任意选择（共振峰附近可加密）
- 抗非线性能力强（窄带不激发谐波）

### 3.3 缺点

- 总时长 = Σ Tk，低频段 Tk 很大
- 不适合快速变化的系统

### 3.4 参数选择

- **频率范围**：覆盖感兴趣的模态（倒立摆典型 0.1~10 Hz）
- **频率密度**：对数分布（低频密、高频疏），共振峰附近加密
- **驻留时长**：`Tk ≥ 5 / (ζωn)`（最弱阻尼模态衰减时间）
- **幅值**：保证 SNR，又不激励出非线性

## 4. 线性扫频 Chirp

```
u(t) = A · sin(2π · (f0·t + (k/2)·t²))
k = (f1 - f0) / T
```

**瞬时频率**：`f(t) = f0 + k·t`，从 `f0` 线性增至 `f1`。

### 4.1 优点

- 单次激励覆盖整段频率
- 配合 Welch 法可直接估计 FRF
- 频段内能量近似平分布

### 4.2 缺点

- 起止有瞬态，建议裁掉首尾各 5%
- 共振峰处能量被"快速扫过"，分辨率受限

### 4.3 参数选择

- **扫频范围 `[f0, f1]`**：覆盖所有关心的模态
- **总时长 `T`**：使扫频速率足够慢，共振峰不被抹平
  ```
  (f1 - f0) / T  <  ζ·ωn²
  ```
- **幅值**：倒立摆这类系统需限制，避免摆角过大触发非线性

## 5. PRBS（伪随机二进制序列）

按 LFSR（线性反馈移位寄存器）生成二进制信号，取值 `±A`。

### 5.1 特性

- 周期性，自相关近似 δ 函数
- 频谱在时钟频率 `1/Δt` 内近似平谱
- n 阶 LFSR 周期 `N = 2ⁿ - 1`

### 5.2 参数选择

- **时钟周期 Δt**：最高辨识频率 `fmax ≈ 1/(3·Δt)`
- **序列长度 N**：覆盖系统调整时间 `N·Δt > 5τ`
- **重复多次平均**：抑制噪声

## 6. 多正弦 Multi-sine

```
u(t) = Σ Ak · sin(2π·fk·t + φk)
```

### 6.1 相位优化

Schroeder 相位使峰值因数最小化：
```
φk = -k(k-1)π / K
```

### 6.2 优缺点

- **优点**：频谱完全可控，峰值因数低（~1.7，远低于扫频的 ~3），单位激励能量下 SNR 最高
- **缺点**：需长周期重复以稳定，设计复杂

## 7. 信号选择决策树

```
是否需要高精度单频点？
├─ 是 → 步进正弦
└─ 否 → 频域辨识还是时域辨识？
        ├─ 频域 → Chirp（快速）或 多正弦（高 SNR）
        └─ 时域 → PRBS
```

## 8. 工程经验

### 8.1 幅值选择
- 太小：SNR 不足，相干低
- 太大：触发非线性，相干在共振峰处骤降
- 经验：从最小可用幅值开始，逐步增大直到相干开始下降，再回退 30%

### 8.2 频率范围
- 上限：取关心最高频率的 2 倍
- 下限：取系统最低模态频率的 1/3
- 共振峰附近：步进正弦需加密，Chirp 需降低扫频速率

### 8.3 预激励
- 启动后先施加 1~2 个完整周期的激励再开始录制
- 让系统进入稳态响应，避免初始瞬态污染
